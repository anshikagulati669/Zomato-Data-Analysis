# 🍽️ Zomato Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Zomato Restaurant Dataset using Python. The objective is to understand restaurant trends, customer preferences, ratings, cuisines, pricing, online ordering, and other important factors through statistical analysis and data visualization.

---

## 🎯 Objectives

* Analyze restaurant distribution across cities.
* Identify the most popular cuisines.
* Study restaurant ratings.
* Explore online ordering trends.
* Analyze restaurant costs.
* Perform correlation analysis.
* Create new features using feature engineering.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset

Dataset: **Zomato_with_city_cuisine.csv**

## ▶️ How to Run

1. Clone this repository
2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Run

```bash
python zomato.py
```
